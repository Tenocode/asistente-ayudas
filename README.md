# Asistente Ayudas

Proyecto para automatizar la búsqueda y análisis de convocatorias de ayudas públicas a partir de documentos PDF.

## Estructura

```
asistente-ayudas/
├── data/
│   └── convocatorias/   # PDFs de convocatorias descargados
├── src/                 # Código fuente del proyecto
├── .gitignore
└── README.md
```

## Requisitos

- Python 3.10+

## Instalación

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
